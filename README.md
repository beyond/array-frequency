# Array::Frequency

## Installation

Add this line to your application's Gemfile:

    gem 'array-frequency'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install array-frequency

## Usage

    > %w[ a b c b b a ].freq
    => {"a"=>2, "b"=>3, "c"=>1}
